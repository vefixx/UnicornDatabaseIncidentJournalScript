# Скрипт обновления журнала инцидентов в базе данных
Скрипт проверяет представления или таблицы с инцидентами и записывает свежие данные в журнал (отдельную таблицу `incidents_journal`)
## Инструкция
### Конфигурация скрипта
Для начала работы необходимо заполнить конфигурационный файл `config.json`, который должен находиться в корневой директории со скриптом.
Структура файла следующая:
```json
{
  "databasePath": "unicorn.sqlite"
}
```
Где:
- databasePath (string): путь до файла базы данных
